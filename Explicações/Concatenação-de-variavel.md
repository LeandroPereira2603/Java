# Concatenação de variável 


```java 

import java.util.Scanner; 
public class Main {
    
    public static void main(String[] args) {
        
        String x = "1"; 
        String  y = "2" ; 
        String  sum = x+y;
        
        System.out.println("\n Numero : "+x+" + "+y+" ");
        
        System.out.println(" Concatenação dos numeros "+sum);
    }
    
}

```
